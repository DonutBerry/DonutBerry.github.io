# a Tasty Age of Moderation!

Donut Berry is a Discord Bot Made for Moderation and Keeping your Donuts steady and stable,

The Bot will try as much as possible to allow you to have as much customizability as you wish

# Documentation

• Features

+ Features

- 1 • Moderation

The Bot's Main Feature is Obviously "Moderation", The Bot will provide you with the tools you will need to moderate a chat, Commands with different functions like,

•Ban/Mute/TimeOut/Kick Commands
•Lock Channels
•Clean last 50 messages in a channel or from a user
•Warnings and Infraction limit
•Moderation Logs
•Message Logs
•Custom Punishment Direct Message
